Spaceship Survival Prediction

Developed a Supervised Machine learning model to predict the survival of passenger traveling, depending upon many features. Applied Logistic Regression and Decision Tree Classifier for the model development and got 90% accuracy on the result data.

Regression model development for prediction of the price of a house in the USA based on different factors. Data cleaning is done with the help of Pandas, for analysis of dependency of features on the price used Seaborn, and normalization of data is done by scikit-learn. Developed multivariate linear regression model and polynomial regression and got 88% accuracy on result data.
